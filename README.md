## Installation

```sh
$ git clone https://github.com/SLP-KBIT/linkus.git
$ cd linkus
$ bundle install -j4 --path vendor/bundle
$ rake db:setup
$ rails s
```

## Requirements

- Ruby 2.2.2
- Ruby on Rails 4.2.3
